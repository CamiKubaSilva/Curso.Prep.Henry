	RESPUESTAS:

	OBJETOS: Es la relacion de un elemento y una propiedad.

	PROPIEDADES: Son caracteristicas de un elemento.

	METODOS: Son funciones que estan dentro de un objeto como propiedad.

	BUCLE 'for...in': Sirve para realizar un bucle pero en un objeto, porque no utiliza numeros para nombrar los objetos sino palabras.

	NOTACION DE PUNTOS: Sirve para llamar a la propiedad de un objeto (es resumida y no se pueden usar variables).

	NOTACION DE BRACKETS: Sirve para llamar a la propiedad de un objeto y si se pueden usar variables.
